# Web Calletano Restaurant

## Descripción
Plataforma web oficial diseñada y desarrollada para "Calletano Restaurant". 
El objetivo principal de este proyecto es la digitalización integral de la oferta gastronómica del establecimiento, fortaleciendo su identidad digital y mejorando significativamente la experiencia del cliente tanto en el local físico como en línea.

## ✨ Características Principales
* **Menú Digital Inteligente:** Implementación de un sistema de acceso rápido al menú mediante códigos QR dinámicos, facilitando la interacción del cliente sin contacto físico.
* **Optimización SEO Técnico:** Estrategias de posicionamiento en buscadores aplicadas desde el código base para incrementar el tráfico orgánico y la visibilidad del negocio en su área local.
* **Diseño Responsivo (Mobile-First):** Interfaz fluida y adaptable que garantiza una experiencia de navegación perfecta en cualquier dispositivo (smartphones, tablets y computadoras de escritorio).
* **Gestión de Contenido Dinámico:** Arquitectura frontend preparada para facilitar actualizaciones de inventario y cambios en el menú en tiempo real.

## 🛠️ Tecnologías Utilizadas
* **Frontend:** HTML5, CSS3, JavaScript
* **Diseño UI/UX:** Enfoque en usabilidad y accesibilidad
* **Optimización:** SEO On-Page, compresión de assets
* **Control de Versiones:** Git, GitHub
